# darthcav.github.io

Darth CAV home page repository

## Customizations

- Website created with [Jekyll](http://jekyllrb.com/)
- Installed Jekyll plugins:
  - github-pages
  - jekyll-github-metadata
  - netrc
  - redcarpet
  - rouge
- After installation `gem update` these packages:
  - jemoji
  - jekyll-mentions
  - jekyll-redirect-from
